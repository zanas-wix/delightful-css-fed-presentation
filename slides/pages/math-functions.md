## Fluid typography with :clamp()

- Math functions can contain `calc()` expressions inside

For example:

- 1rem as a minimum
- 1rem + 5vw as a fluid value
- 2.5rem as a maximum

`rem` component of this equation should be at least 1rem, so that users can scale it effectively

::window::

```css
.fluid-heading {
    font-size: clamp(
        1rem,
        2vw + 1rem,
        2.5rem
    );
}
```
