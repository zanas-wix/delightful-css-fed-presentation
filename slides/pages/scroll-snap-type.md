## Scroll snap possibilities

<v-clicks>

- Carousels
- Full screen sections (aka slides)
- Photo galleries
- Mobile-app interfaces (Tiktok clone)

</v-clicks>

::window::

```css
/* No-JS carousel */
.scroll-container {
    scroll-snap-type: x mandatory;
    overflow-x: scroll;
}

.scroll-item {
    scroll-snap-align: start;
}
```
